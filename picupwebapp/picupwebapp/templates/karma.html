Karma system explained

What is the karma?
Karma is a simple reputation system. Users may be awarded or punished, depends on their actions inside the PicUp platform.
Is it realtime?
No. Karma calculaction happens every 10 minutes. Around :)

Accounting - Rules

Awards
User's points
  • +200 - pioneer (id below 10)
  • +100 - very early user (below 20)
  • +50 - early user (below 100)
  • +25 - adopter (below 1000)
Picture's points
  • +5 - upload of the photo
  • +3 - vote on the photo
  • +1 - set photo in the gallery
  • +100 - launch, link in your profile and verify by stuff application which uses an api
  • +250 - Idea to improve the service which will be implemented
  • +25 - Karma idea
Users points will be cumulated.
Punishments
  • -1 Creating empty gallery
